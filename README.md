- 🌙 **Dark Mode / Light Mode Toggle**
- 🎨 **Custom SCSS Styling**
- 🖥 **Fully Responsive Design**
- 📁 **Projects Section with Live Links**

- 📌 **GitHub Pages Deployment (samgee.gamwise.me)**
